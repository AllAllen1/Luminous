# Luminous
видеохостинг 

📄 Паспорт проекта: Luminous
1. Проблема, которую мы видим (The Pain)
Сегодняшние видеохостинги (YouTube, Vimeo и др.) превратились в «золотые клетки».

Изоляция контента: Сложно легально и быстро скачать видео для офлайн-просмотра или монтажа.

Избыточная цензура и реклама: Площадки перегружены алгоритмами, которые мешают авторам и раздражают зрителей.

Зависимость от интернета: Пользователи не владеют контентом, который смотрят; если видео удалят или пропадет сеть — доступ потерян.

2. Цель: Для чего проект создаётся (The Goal)
Создать среду видеохостинга для пользователей без потерь данных.

Миссия: Создать экосистему Luminous, где видео — это свободный актив, а не просто поток данных.

Техническая цель: Разработать платформу с эффективным транскодированием видео и системой генерации прямых ссылок на загрузку в разных форматах.

3. Стороны: Кому оно надо (Stakeholders)
Контент-мейкеры: Авторы, которым важно, чтобы их видео можно было использовать как образовательный или исходный материал.

Студенты и преподователи: Те, кому нужно сохранять лекции и материалы для работы без интернета.

4. Границы: Что изучаем и что мешает (Scope & Risks)
Что нужно изучить (Focus):

Протоколы передачи данных: Как быстро отдавать тяжелые файлы.

Кодеки (FFmpeg): Как сжимать видео без потери качества для экономии места на сервере.

Что может помешать (Obstacles):

Стоимость хранения: Видео требует огромных дисковых пространств.

Абузы (Злоупотребления): Загрузка пиратского или запрещенного контента.

Нагрузка на канал: Высокий трафик при одновременном скачивании видео тысячами пользователей.

5. Где расположить (Deployment)
Для старта проекта Luminous оптимально использовать гибридный подход:

Backend & Frontend: Облачные VPS (например, Timeweb, Selectel или AWS) — для быстрой работы интерфейса.

Хранилище видео: S3-совместимые хранилища (Object Storage). Это дешевле и масштабируемее, чем обычные жесткие диски.

CDN (Content Delivery Network): Обязательно расположить узлы раздачи поближе к целевой аудитории, чтобы видео «летало».
